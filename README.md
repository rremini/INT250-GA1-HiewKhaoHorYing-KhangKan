# KhangKan
<img width="4341" height="1320" alt="khangkan-white-logo" src="https://github.com/user-attachments/assets/c9869b7a-81f9-4fcc-b65c-826f0d89fb33" />

Digital Safe Space: A web application designed to provide students with a secure environment to manage stress, pressure, and various mental health challenges. These issues often stem from diverse sources, including social pressures, family conflicts, violence, and financial difficulties, all of which can lead to significant anxiety. Our primary focus is to promote a correct understanding of mental health and ensure the platform is easily accessible to everyone in need.

### Objectives
* To build a functional and visually appealing website using Tailwind CSS
* To practice and apply CSS frameworks in collaborative web development using Agile methodology
* To demonstrate understanding of responsive design and modern UI components.
